# 🤖 Project Astra

Project Astra is my personal journey to becoming an Enterprise AI Engineer by building a production-quality AI assistant from scratch.

Instead of following tutorials, I am designing and implementing every component myself—from the backend APIs to the AI integration, frontend, prompt engineering, and eventually AI agents.

The long-term vision is to build an intelligent assistant capable of searching the web, using tools, maintaining memory, and acting autonomously.

---

# ▶️ Running the Project

## Start Backend

```bash
start cmd /k "py -m uvicorn backend.app:app --reload"
```

---

## Start Frontend

```bash
start cmd /k "py -m streamlit run frontend/app.py"
```

---

## Run Complete Application

Simply execute:

```text
run.bat
```
